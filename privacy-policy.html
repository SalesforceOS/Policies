SFOS Core

Privacy Policy

Last updated: January 29, 2025

SFOS Core is a browser extension designed to help Salesforce administrators and developers work more productively. We are committed to protecting your privacy and being transparent about how our extension works.

Our Commitment

SFOS Core operates entirely within your browser. Your Salesforce data is never transmitted to external servers, and we do not collect, store, or share any personal information.

Data We Access

To provide its functionality, SFOS Core accesses the following data from your Salesforce org:

Data Type Purpose Stored
Salesforce Record Data Display records, clone records, analyze relationships No
Object Metadata Show field information, object relationships Local cache only
User Preferences Remember your settings and saved recipes Local storage only
Session Cookie Authenticate API requests to Salesforce No

Data We Do NOT Collect

SFOS Core does not collect, store, or transmit:

  • Personally identifiable information (names, emails, addresses)
  • Salesforce usernames or passwords
  • Browsing history or activity outside Salesforce
  • Financial or payment information
  • Location data
  • Health information
  • Any data to external servers or third parties

Local Storage

SFOS Core stores certain data locally on your device using Chrome's built-in storage APIs. This data never leaves your browser and includes:

  • User Preferences: Theme settings, display options, feature toggles
  • Saved Recipes: Data generation configurations you create and save
  • Metadata Cache: Object and field information cached for performance
  • Field Configurations: Custom field generator settings

Your Control

You can clear all locally stored data at any time through the extension settings or by removing the extension from Chrome.

Salesforce API Access

SFOS Core uses your existing Salesforce session to make API calls. Here's how it works:

  • The extension reads your Salesforce session cookie to authenticate API requests
  • API calls are made directly to your Salesforce org's servers
  • Your session credentials are never stored, logged, or transmitted elsewhere
  • The extension only works when you're logged into Salesforce

Google Sheets Integration (Optional)

SFOS Core offers optional Google Sheets integration for importing data and team collaboration. This feature is entirely opt-in.

  • Only activated when you explicitly choose to connect your Google account
  • Uses standard OAuth 2.0 authentication
  • Only accesses Google Sheets you specifically select
  • You can disconnect your Google account at any time
  • No Google account data is stored beyond the OAuth token required for the connection

Third-Party Services

SFOS Core does not use any third-party analytics, tracking, or advertising services. The extension does not communicate with any servers other than:

  • Your Salesforce org (for API calls using your session)
  • Google APIs (only if you enable Google Sheets integration)

Data Security

We implement the following security measures:

  • All Salesforce API communications use HTTPS encryption
  • No sensitive data is stored in plain text
  • The extension only activates on Salesforce domains
  • No remote code execution - all code is bundled within the extension

Children's Privacy

SFOS Core is a business productivity tool intended for Salesforce professionals. It is not directed at children under the age of 13, and we do not knowingly collect information from children.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. Significant changes will be communicated through the extension update notes.
